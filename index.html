啟動專案

Git

  1. git -v
  2. npm install hexo-deployer-git –save
  3. 到 github 上 new 一個專案
  4. 到 _config.yml 找到 deploy 並把下方的資訊複製貼上 repo 是指 git 的網址
    repo 的網址是 git 專案的網址
  1. 到 _config.yml 大概是第 16 行
    這是指 網址 要打上去 https://使用者.github.io/專案名稱
  1. git clean
  2. git deploy
  3. 就大功告成了
